Путь Петра — финальная версия 1:1

Как открыть:
1. Распакуйте архив.
2. Откройте index.html в браузере.
3. Нажмите «Запустить».

Что изменено:
- Убрана лишняя плашка «Путь раскрывается под песню».
- Кнопка «Пауза» / «Продолжить» теперь видна и работает.
- Версия только 1:1.
- Раскрытие идет под музыку 46 секунд.
- Если файл музыки имеет другую точную длину, анимация подстроится по длительности аудио.
- После завершения картинка не закрывается: она остается полностью открытой, чтобы ее можно было спокойно рассмотреть.
- Убрана анимация пути по трубе.
